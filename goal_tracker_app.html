Troubleshooting Tips

⚠️ IMPORTANT - File Protocol Issue:

You're running this from a file:// URL. WebSocket connections require an HTTP server.

Quick Solutions:

  • Python: python -m http.server 8000
  • Node.js: npx http-server
  • VS Code: Install "Live Server" extension
  • Chrome: Launch with --allow-file-access-from-files flag

Then access via http://localhost:8000

1. Check Streamer.bot WebSocket Server:

  • Open Streamer.bot → Servers/Clients → WebSocket Server
  • Ensure "Auto Start" is enabled
  • Default port should be 8080
  • Click "Start" if it's not running

2. Firewall/Network:

  • Allow Streamer.bot through Windows Firewall
  • Try disabling antivirus temporarily
  • Check if port 8080 is blocked

3. Browser Issues:

  • Use Chrome/Edge (best compatibility)
  • Allow mixed content if using HTTP
  • Check browser console for errors (F12)
